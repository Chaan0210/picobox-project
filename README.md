# PICOBOX 인터넷 영화예매 시스템
데이터베이스설계 프로젝트로 PICOBOX 서비스의 front-end 코드입니다. React.js를 기반으로 영화 조회부터 예매, 결제, 리뷰 등의 전 과정을 웹에서 제공합니다.
사이트 주소: <https://picobox.vercel.app/> (현재는 서버 중단)

## 주요 기능
- 회원 가입 및 로그인, 비회원 예매 지원
- 영화/배우 검색 및 상세 정보 조회
- 상영 스케줄 확인 후 좌석 선택과 예매 진행
- 토스페이먼츠 테스트 API 연동 결제 및 환불 처리
- 포인트 적립과 사용 내역 관리
- 리뷰 작성과 추천 기능
- 관리자 페이지를 통한 영화/배우/상영 관리 및 회원 관리
- report/데이터베이스설계_최종보고서.pdf 참고

## 개발 환경
- **Frontend**: React.js, React Router DOM, styled-components
- **Database**: Oracle 19c
- **Backend**: Spring Boot 3.4.5 (Java 21)
